URL Shortener made with Python, Bottle.py, Redis and gevent

  • stats and url data stored with Redis, using HASH datatype
  • monkey patch and server adapter for WSGI and GEvent
  • supervisord config file to keep running
  • included: bookmarklet, stats page, API

INSTALL

install python-redis (thru apt, or easy_install redis) install python-gevent (or easy_install gevent - which is not good because it needs gcc installed)
